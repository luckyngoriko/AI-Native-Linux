# Executive Summary — Rev.2

Status: `PARTIAL` (rev.2 contracts are landing incrementally)

## Purpose of rev.2

Rev.1 established the AIOS vision and the layer model. Rev.2 turns that vision into contract-grade specifications: schemas, state machines, error models, and verification rules that an implementer can build against without further interpretation.

- **S3.1 (contract):** Evidence log architecture. - **S4.1 (contract):** AIOS-FS namespace layout. Scope: closed namespace, default-deny cross-group, recovery-only admin path, virtual inbox views, catalog-versioned determinism.
- **S3.2 (contract):** Sandbox composition language. Refined 2026-05-08 with deltas D1–D12: proto IDL with `aios.sandbox.v1alpha1` package and `SandboxComposer` gRPC service (`ComposeProfile`/`ValidateProfile`/`ApplyProfile`/`RevokeProfile`/`ExplainComposition`/`GetComposerInfo`);
