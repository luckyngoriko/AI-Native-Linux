# Code of Conduct

AI-Native Linux is an open technical project. Participation should stay professional, specific, and focused on improving the architecture, security model, documentation, and implementation path.

Expected behavior:

- be respectful and direct
- criticize designs and code, not people
- provide concrete technical reasoning
- disclose security-sensitive details responsibly
- keep discussions relevant to the project

Unacceptable behavior includes harassment, personal attacks, threats, spam, and publishing private information or secrets.

Maintainers may close, edit, or remove issues, comments, or pull requests that violate these expectations.
