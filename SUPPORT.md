# Support

AI-Native Linux is currently specification-first. There is no installable operating system or production runtime yet.

Use GitHub issues for:

- unclear specification text
- missing contracts
- contradictions between layers
- implementation questions
- proposed safety improvements
- documentation or site issues

Use the security process in [SECURITY.md](SECURITY.md) for security-sensitive design findings.

Do not post credentials, private keys, production hostnames, customer data, or third-party secrets in public issues.
